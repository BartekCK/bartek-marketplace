# Software Development

Expert backend, frontend, and testing agents with TypeScript/Node.js specialization.

## Agents

| Agent | Specialization |
|-------|---------------|
| **backend-software-developer** | TypeScript/Node.js, SOLID, OOP, design patterns, clean architecture, API design, database patterns |
| **frontend-software-developer** | TypeScript/React, Tailwind CSS, Radix UI, shadcn/ui, responsive design, accessibility |
| **tester** | Unit, integration, and e2e tests with Vitest/Jest, Given/When/Then structure, parameterized tests |

Backend and frontend agents enforce strict typing (no `any`/`undefined`), proper error handling, and always lint after finishing. The tester agent collaborates with both by verifying their implementations.
